# Damage Report Form

Mobile-friendly web form for warehouse staff to report equipment damage in real time.

## Live App

**URL:** `https://lavertoncoldStorage.github.io/damage-report-form/`

## Features

- ✅ **Cascading dropdowns** — Equipment Type auto-filters Equipment ID
- ✅ **Camera capture** — Mobile-optimized photo upload (iOS & Android)
- ✅ **LCS branding** — Professional styling with company logo
- ✅ **Mobile-responsive** — Works on phones, tablets, desktops
- ✅ **Form validation** — Clear error messages
- ✅ **All required fields** — Site, Equipment Type, Equipment ID, Issue Description, Photo, Submission Date

## How to Use

1. Open the form on any device (phone, tablet, or desktop)
2. Select Site and Equipment Type
3. Equipment ID options automatically filter based on Equipment Type
4. Fill in Issue Description and take a photo with the camera button
5. Click "Submit Report"
6. Form clears and data is stored locally on the device

## Mobile Tips

- **Add to home screen:** Open the form, tap menu, select "Add to Home Screen"
- **Camera:** Tap the photo button to use your device's camera
- **Offline:** Form works offline and syncs when connection is restored
